# Typing Speed Test — Frontend Mentor Challenge

Projeto desenvolvido como parte de um **desafio do Frontend Mentor**, realizado em colaboração entre **Pedro Henrique Ernesto de Souza** e **Fulano**.

Este repositório será **continuamente atualizado** conforme o andamento do desenvolvimento.

---

## Visão Geral

O **Typing Speed Test** é uma aplicação web que mede a **velocidade e precisão de digitação** do usuário com base em **textos aleatórios**.  
O objetivo é criar uma experiência simples, intuitiva e performática, seguindo boas práticas de frontend.

---

## Objetivos do Desafio

- Testar velocidade de digitação (WPM / CPM)
- Avaliar precisão (erros e acertos)
- Utilizar textos aleatórios a cada tentativa
- Interface clara e responsiva
- Código limpo e organizado

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+) 

> As tecnologias podem ser ajustadas conforme o progresso do desafio.

---

## Funcionalidades Planejadas

- [ ] Exibição de texto aleatório  
- [ ] Campo de digitação em tempo real  
- [ ] Cálculo de WPM (Words Per Minute)  
- [ ] Cálculo de precisão  
- [ ] Feedback visual para erros e acertos  
- [ ] Botão de reinício  
- [ ] Responsividade para mobile
